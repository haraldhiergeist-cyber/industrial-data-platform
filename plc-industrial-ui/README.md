# PLC-Industrial-UI
A modern Angular-based dashboard for real-time visualization of industrial PLC data.
This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.

## Overview
The PLC Industrial UI displays live PLC data via a REST API (plc-query-service).

## Features
- Real-time dashboard (polling)
- Angular + PrimeNG UI
- OpenAPI generated client
- Dark theme

## Architecture
PLC → Collector → Kafka → Redis → Query Service → Angular UI
